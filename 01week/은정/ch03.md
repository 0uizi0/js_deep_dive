# 핵심 내용 정리

## 3.1 자바스크립트 실행 환경

- 브라우저는 Node.js와 용도가 다르다
    - 브라우저 : HTML, CSS, JS를 실행해 웹 페이지를 브라우저 화면에 렌더링하는 것이 주된 목적
    - Node.js : 브라우저 외부에서 자바스크립트 실행 환경을 제공하는 것이 주된 목적
